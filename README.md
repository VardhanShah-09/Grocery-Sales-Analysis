# Grocery-Sales-Analysis
Grocery Sales Analysis Dashboard using Python and Tableau with RFM Customer Segmentation and Business Insights

Problem Statement:
Retail businesses generate large volumes of sales data, but it is difficult to extract meaningful insights such as sales performance, profit trends, and customer behavior from raw data. Businesses need a dashboard to track key performance indicators and support data-driven decision-making.

Project Objective:
The objective of this project is to analyze grocery store sales data and build an interactive dashboard to understand:
Sales performance
Profit analysis
Customer purchasing behavior
Monthly sales trends
Customer segmentation using RFM analysis

Tools & Technologies Used:
Python (Pandas, Matplotlib)
Tableau
Jupyter Notebook
Excel/CSV Dataset

Project Steps:
Data Cleaning using Python
Data Preprocessing
RFM (Recency, Frequency, Monetary) Analysis
Customer Segmentation
Data Visualization
Dashboard Creation in Tableau

Dashboard Features:
KPI Cards (Total Sales, Total Profit, Total Customers)
Sales by Category
Sales by Store
Monthly Sales Trend
Discount vs Sales Analysis
Top 10 Products
Customer Segmentation (RFM)
Interactive Filters (Store, Year)

Key Insights:
Some product categories generate higher sales than others.
Higher discounts do not always lead to higher sales.
A small group of customers contributes a large portion of revenue.
Sales vary across months showing seasonal trends.
Some stores perform better compared to others.

Conclusion:
This project helps in understanding sales performance, customer segmentation, and profit analysis, which helps businesses make better strategic decisions.
